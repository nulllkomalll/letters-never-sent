# Letters Never Sent

A digital archive for words people never got to say.

Write letters to:
- Your younger self
- Your future self
- A loved one
- An old friend
- Someone you've lost

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Node.js (planned)
- Express.js (planned)
- PostgreSQL (planned)

## Features

- Anonymous letter publishing
- Future Vault
- Emotional letter categories
- Personal profiles
- Search & discovery
