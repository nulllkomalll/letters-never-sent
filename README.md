# Letters Never Sent

A digital archive for words people never got to say.

Write letters to:
- Your younger self
- Your future self
- A loved one
- An old friend
- Someone you've lost

---

## Project Vision

Letters Never Sent is a digital archive for words that were never spoken.

Rather than functioning as a messaging platform, it provides a space for emotional expression, reflection, and storytelling.
